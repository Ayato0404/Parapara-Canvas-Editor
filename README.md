# parapara-projection-mapping

## 準備
### 必要なもの
- node.js
- bower (npm install -g bower)

### モジュールをインストール

```
npm install
bower install
```

### webpack でビルド

```
webpack
```

### webpack-dev-server を使ってビルド

```
npm run build-dev-server
```
